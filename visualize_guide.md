# Agent 可视化生产指令

## 全局输入约束

- **数据格式**：按时间升序排列的字典列表 `list[dict]`，键名同表头。
- **时间处理**：仅用原生 `datetime` 解析 `timestamp`。采样间隔 `Δt` 由相邻两条记录的实际时间差（秒）动态计算，不预设固定值。
- **负值处理**：所有 `ΔX = X(t) - X(t-1)` **保留原值**（允许负数），绘制时正常显示负向柱状或折线下跌。
- **视频时长**：外部输入常量 `D`（单位：秒）。

---

## 一、全局派生指标算法（所有图共用）

对每条记录 `i`（从 `1` 到 `N-1`），计算以下增量字典：

```
Δt = (timestamp_i - timestamp_{i-1}).total_seconds()

Δviews    = views_i - views_{i-1}
Δlikes    = likes_i - likes_{i-1}
Δcoins    = coins_i - coins_{i-1}
Δfavorites= favorites_i - favorites_{i-1}
Δdanmaku  = danmaku_i - danmaku_{i-1}
Δshares   = shares_i - shares_{i-1}
Δreply    = reply_i - reply_{i-1}
Δrank     = rank_i - rank_{i-1}   (可为正负，正值表示排名下跌)
```

**积分期望播放量（梯形积分法）**：

```
expected_delta_i = (online_{i-1} + online_i) / 2 * Δt / D
```

## 二、必须生产的图表清单（共 7 张）

### 图1：核心总量趋势图

- **类型**：双轴折线图。
- **左轴**：`views`（累计值）。
- **右轴**：`likes`、`coins`、`favorites`（累计值）。
- **横轴**：`timestamp`。

### 图2：在线与弹幕脉冲图

- **类型**：面积图 + 柱状图叠加。
- **面积**：`online`（实时在线人数）。
- **柱状**：`Δdanmaku`（每条间隔的弹幕增量，允许负值则柱向下）。

### 图3：加权质量指数（算法健康度）

- **类型**：折线图。
- **指标**：`(Δcoins * 3 + Δfavorites * 2 + Δlikes * 1) / Δviews`。
- **注意**：若 `Δviews <= 0`，该点跳过不绘。

### 图4：三连转化比值（分组柱状图）

- **类型**：按时间分组，每组3根柱子。
- **柱子1**：`Δlikes / Δviews`
- **柱子2**：`Δcoins / Δviews`
- **柱子3**：`Δfavorites / Δviews`
- **注意**：若 `Δviews <= 0`，该时间点整组跳过。

### 图5：观看留存深度（VDR）柱状图（核心原创）

- **类型**：柱状图（基准线绘制在 `y=1.0`）。
- **指标**：`VDR = Δviews / expected_delta`。
- **注意**：`expected_delta` 可能为 0 或极小，此时 `VDR` 标记为 `Inf` 或 `-Inf`，该点跳过不绘。负的 `Δviews` 产生负向柱状图。

### 图6：即时平均停留时长追踪线

- **类型**：折线图 + 水平红线（`y=D` 视频总时长）。
- **指标**：`Avg_Stay = ∫(online)dt / Δviews`，其中 `∫(online)dt = (online_{i-1} + online_i)/2 * Δt`。
- **注意**：若 `Δviews <= 0`，该点跳过不绘。

### 图7：裂变传播散点回归图（超前预测）

- **类型**：二维散点图。
- **X轴**：`expected_delta / Δt`（即 `(online_{i-1} + online_i) / (2 * D)`，单位：等效完播数/秒）。
- **Y轴**：`Δviews / Δt`（实际播放增速，单位：次/秒）。
- **气泡大小/颜色**：映射至 `Δshares` 的绝对值（若 `Δshares` 为负，取 `0` 或显示空心圈）。
- **辅助线**：绘制 `Y = X` 灰色虚线。

## 三、进阶可选图（数据非空时绘制）

### 图8：排位弹性系数

- **类型**：面积折线图（Y轴倒序，数值越小排名越高）。
- **指标**：`Elasticity = Δviews / |Δrank|`。
- **注意**：若 `Δrank == 0`，该点跳过。`his_rank` 作为水平参考虚线标注在全图。

### 图9：分享-播放时滞归因（错位叠加）

- **类型**：双轴折线图。
- **操作**：将 `Δshares` 序列整体**向前平移 1 个索引**（即错位一个采样间隔），再与 `Δviews` 绘制于同一图中，观察重合度。
- **可选扩展**：若数据点 > 20，计算平移 0、1、2 个索引后的皮尔逊相关系数（手算协方差/标准差），取相关系数最高的平移量标注在图上。
